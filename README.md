# Java ToDo App

Javaの学習の一環として作成した、コンソール版のToDoアプリです。

## 作成した目的
これまで学んだ ` if ` ` while ` などを使って、実際に動くアプリを作る練習として作成しました。
最初は1件だけ表示する形から始め、少しずつ改良して、ToDoを追加して一覧表示できる形にしました。

## アプリの内容
- ToDoを1件ずつ入力できる
- 入力した内容を一覧に追加できる
- 空入力は無効として再入力を促す
- 続けて追加するか、終了するかを選べる

## 学習ポイント
- 繰り返し処理
- 条件分岐
- 文字列の連結
- コンソール入力
- 小さく作って少しずつ改良する流れ
